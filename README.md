# Rina Games

## About Project

Rina Games is a collection of mini games created using web technologies.

This project focuses on frontend development and interactive user experience.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Multiple mini games
- Interactive UI
- Responsive design
- User friendly interface

## Games Included

- Candy crush
- Socobon
- tetris
- chess
- breackout
- bubble shooter
- ping pong,etc.

## How To Run

1. Download the project
2. Open index.html
